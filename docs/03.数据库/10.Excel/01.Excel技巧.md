## Excel 使用

### 相同值的单元格合并

使用功能：分类汇总、条件定位

1. 选中需要合并的列：点击【数据】 -> 【分类汇总】 -> 【确定】
2. 选中上一步生成列的部分行：【F5/Ctrl+G】 -> （定位条件）勾选【空值】 -> 【确定】 -> 【开始】 -> 【合并居中】
3. 选中任意单元格：点击【数据】 -> 【分类汇总】 -> 【全部删除】
4. 选中格式正确的空白列：点击【开始】-> 【格式刷】 刷好我们需要合并的列
5. 选中空白列：右键删除
