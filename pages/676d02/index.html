# зӯ–з•ҘжЁЎејҸпјҲStrategy Patternпјү

зӯ–з•ҘжЁЎејҸе®ҡд№үдәҶз®—жі•ж—ҸпјҢеҲҶеҲ«е°ҒиЈ…иө·жқҘпјҢи®©е®ғд»¬д№Ӣй—ҙеҸҜд»Ҙдә’зӣёжӣҝжҚўпјҢжӯӨжЁЎејҸи®©з®—жі•зҡ„еҸҳеҢ–зӢ¬з«ӢдәҺдҪҝз”Ёз®—жі•зҡ„е®ўжҲ·гҖӮ

# йёӯеӯҗйЈһе’ҢеҸ«

дёәдәҶз»ҷйёӯеӯҗеўһеҠ йЈһиЎҢиғҪеҠӣ

й—®йўҳпјҡеңЁи¶…зұ»еўһеҠ ж–№жі•еҜјиҮҙж©Ўзҡ®йёӯд№ҹдјҡйЈһдәҶ и§ЈеҶіж–№жЎҲпјҡеңЁеӯҗзұ»дёӯиҰҶзӣ–жҺүзҲ¶зұ»ж–№жі•пјҢеҸҲдјҡеҜјиҮҙе…¶д»–й—®йўҳпјҲдёҖж—ҰеҲӣе»әж–°зҡ„йёӯеӯҗзұ»пјҢе°ұеҝ…йЎ»жЈҖжҹҘйЈһиЎҢе’ҢеҸ«зҡ„ж–№жі•пјү

дҪҝз”ЁжҺҘеҸЈе‘ўпјҢдјҡеҜјиҮҙ

  1. д»Јз ҒдёҚиғҪеӨҚз”Ё
  2. йЈһиЎҢеҠЁдҪңжңүеҢәеҲ«пјҢе°ұиҰҒжҜҸдёӘзұ»зҡ„ж–№жі•йғҪжңүдёҚеҗҢзҡ„е®һзҺ°

й’ҲеҜ№жҺҘеҸЈзј–зЁӢ зңҹжӯЈеҗ«д№үдёәй’ҲеҜ№и¶…зұ»зј–зЁӢгҖӮеҚіAnimal animal = new Dog()зҡ„еҪўејҸ дјҳзӮ№пјҡ

  1. йЈһиЎҢе’ҢеҸ«зҡ„еҠЁдҪңеҸҜд»Ҙиў«е…¶д»–еҜ№иұЎеӨҚз”Ё
  2. еҸҜд»Ҙж–°еўһиЎҢдёәпјҢдёҚдјҡеҪұе“ҚеҲ°иЎҢдёәзұ»пјҢд№ҹдёҚдјҡеҪұе“ҚеҲ°йёӯеӯҗзұ»

зјәзӮ№пјҡжһ„йҖ еҷЁе®һдҫӢеҢ–еҜ№иұЎпјҢжҳҜеҜ№е…·дҪ“е®һзҺ°зј–зЁӢпјҢжңүеҫ…ж”№иҝӣ

ж”№иҝӣпјҡ еҲӣе»ә FlyBehaviorгҖҒQuackBehavior зҡ„ getгҖҒset ж–№жі•пјҢдҪҝиҝҗиЎҢж—¶еҸҜд»ҘеҠЁжҖҒзҡ„и®ҫе®ҡе…¶иЎҢдёәпјҢеҚіеңЁеҲӣе»әйёӯеӯҗеҜ№иұЎж—¶дј е…ҘиЎҢдёәеҜ№иұЎжқҘи®ҫзҪ®е…¶иЎҢдёәгҖӮпјҲжңүзӮ№зұ»дјјдәҺеҮҪж•°зј–зЁӢйҖҡиҝҮеҸӮж•°дј е…ҘдёҖж®өйҖ»иҫ‘зҡ„ж„ҹи§үпјҹпјҹпјҹпјү

# е°ҸжёёжҲҸпјҡеҠЁдҪңеҶ’йҷ©жёёжҲҸ

жңүжёёжҲҸи§’иүІе’ҢдҪҝз”ЁжӯҰеҷЁиЎҢдёәзҡ„зұ»пјҢжҜҸдёӘи§’иүІдёҖж¬ЎеҸӘиғҪдҪҝз”ЁдёҖз§ҚжӯҰеҷЁпјҢдҪҶжҳҜеҸҜд»ҘеңЁжёёжҲҸиҝҮзЁӢдёӯжҚўжӯҰеҷЁгҖӮ
